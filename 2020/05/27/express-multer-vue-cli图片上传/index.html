express + multer + vue-cli

multer 是 node 的一个中间件，处理文件上传

注意： multer 只能处理 multipart/form-data 格式数据
因此要求 前端 使用 form 需要使用enctype 或 修改 请求头 headers

express 后台部分

multer 提供磁盘储存 可以对文件进行更多控制

multer 提供多文件上传、但文件上传，官网有详细解释。
此处 使用 single 单文件上传 . 单文件以及文本域 见 官方
传送门： multer 插件 上传文件

指定 文件上传后 保存的文件夹 并进行接口编写

注意： upload.single(‘xxx’) 此处的值需要与前端表单元素的name 属性 一致！否则 会报错： unexpected field 或 undefined
