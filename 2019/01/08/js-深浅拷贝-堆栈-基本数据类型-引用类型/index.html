堆和栈 与 js 数据类型

栈(stack)为自动分配的内存空间,由系统释放;
堆(heap) 动态分配的内存, 大小不定, 不会自动释放;

js 数据类型分为：基本数据类型 & 引用数据类型

  1. 基本数据类型 主要存放在栈内存中, 栈内存中的数据, 按值存放, 可以直接访问, 因此也可以称为 值类型,

  2. 数据在内存中创建后,开辟独立的内存空间,

基本数据类型的值不可变<

基本数据类型比较的是值, 只要值相等就认为其相等

赋值 与 浅拷贝
