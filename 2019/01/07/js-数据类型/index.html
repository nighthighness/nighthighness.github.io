js 数据类型分类

js数据类型分为原始类型 和 引用类型，原始类型也被称为值类型

值类型

字符串(String)、数字(Number)、布尔(Boolean)、空(Null)、未定义(Undefined)

引用数据类型

对象(Object)、数组(Array)、函数(Function)

强制转换

① Number() // 强制转换为nubmer
② String() // 强制转换为string
③ Boolean() // 强制转换为boolean
