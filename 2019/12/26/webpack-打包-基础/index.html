weback 是什么

根据官方文档说明：
webpack 是一个现代 JavaScript 应用程序的静态模块打包器(module bundler）,
webpack将处理应用程序， 将应用程序以 依赖关系 的形式 将模块 打包成 一个 或 多个 bundler
