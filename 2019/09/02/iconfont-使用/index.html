阿里巴巴图标库使用

首先 ，有一个 账号

可使用 github 账号 授权登陆

创建项目

将图标 加入 购物车

添加至项目

打开右上角 购物车 图标
并 添加至项目
此处， 如果没有项目， 可创建新的项目

生成动态链接 及 class 类名

如图：在项目中， 可生成动态链接 ，及class 类名，需要主要的是，每次新添加
图标后，需要更新 链接

代码使用
<!DOCTYPE html>
<html>
<head>
    <title></title>
    <link rel="stylesheet" type="text/css" href="http://at.alicdn.com/t/font_963337_342cp10mx9f.css">
</head>
<body>
    <div>
        <i class="iconfont icon-cat"></i>
    </div>
</body>
</html>
